Detailed Description

  • This file defines the BLT C-API

The API is primarilty used in the stand-alone solver 'run_blt' and to define high level haskell bindings to libblt (part of the 'blt' Haskell package).

Add the assumption l <= 'expr' <= u to the assumption stack.

In the assumption, l = 'n1'/'d1' and u = 'n2'/'d2'. A basic satisfiabilty check is done to ensure that l <= u.

Return values: 

STATUS_OK
STATUS_ERROR (out of memory)
CHECK_UNSAT  (assumption is trivially UNSAT)

CContext blt_init ( int  enable_yices)

Initialize a new problem context.

If 'enable_yices' is non-zero, then the Yices library will be loaded dynamically as needed. The client is responsible for calling 'blt_free' when the context is no longer needed.
