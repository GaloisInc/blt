Variable Documentation

int load_yices_c

A reference counter indicating whether yices shared library has been loaded or not and by how many clients.

load_yices_c == -1: loading yices failed for some reason load_yices_c == 0: yices is not currently loaded load_yices_c >= 1: yices has been successfully loaded this number of times
