Detailed Description

Context for the problem being solved; It records variables and expressions and provides an API for enable DPLL-style search.

Context objects are only created using the default initializer. They are updated using mkVar, assume, check, and backtrack.

int Context::check ( )

Check the satisfiability of the current set of assumptions.

If the current formula is satisfiable, return BLT_CHECK_SAT. If the current formula is unsatisfiable, return BLT_CHECK_UNSAT. The function can also return BLT_STATUS_INPUT_ERROR, and BLT_STATUS_ERROR.
